¿Que es el Diseño UX/UI?

Conocido también como UX Design (User Experience Design), el diseño UX es un sistema desarrollado para crear productos que resuelvan las necesidades de los usuarios finales cubriendo todos los aspectos, desde el tema gráfico y la interfaz, hasta la interacción del usuario con el producto final, de tal forma que la experiencia de usuario sea la mejor posible. Cualquier producto o servicio necesita de una buena experiencia de usuario. De hecho, según la consultora Nielsen, si el UX de un sitio web es ineficaz -por ejemplo si la página demora en cargarse o su navegación es confusa– el usuario abandona el sitio en un promedio de 15 segundos; y 9 de cada 10 usuarios que tienen una mala experiencia en una página web no regresan a ella. Una buena experiencia de usuario va más allá de una página web. Muchos usuarios utilizan sus dispositivos móviles para consultar información, navegar por una página e incluso hacer compras en plataformas de comercio electrónico. Por esta razón, el diseño UX debe ser responsive, es decir, debe ajustarse sin problemas a cualquier tipo de pantalla, navegador y sistema operativo.
